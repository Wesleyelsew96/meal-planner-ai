# Meal Planner Workspace

This repository tracks the **meal-planner-ai** project that powers a rotation-based meal suggestion engine and browser UI.

- Main app code lives under `meal-planner-ai/`
- Other projects (`ai-interview-coach/`, `lyrics-generator/`) are separate repositories kept in the same workspace but ignored by this repo

See `meal-planner-ai/README.md` for setup, usage, and roadmap details.
